OUR PROJECTS

Nengine

The Nengine is going to be our biggest projects yet. It is going to be a usable, general-purposed 2D game engine built almost entirely from scratch with a usable editor and Vulkan renderer. As of right now, it is still extremely early in development, to the point in which we don't even have a Vulkan logical device setted up. Anyways, you can check it out here.

C++++

Not to be confused with a language by Microsoft with a similar name, C++++ is our first attempt at trying to create a brand-new language with a brand-new foundation that could potentially replace C++. Despite the naming scheme, it is not backwards compatible with either C nor C++. It features memory-safety, modular code organization, modern syntax, and more while offering the performance and low-level control demanded by C++ developers. We currently do not have a reference implementation yet, but you can view the (rather incomplete) specification here.
